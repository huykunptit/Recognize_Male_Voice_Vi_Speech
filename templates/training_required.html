Training Required

Dữ liệu chưa được training để so sánh giọng nói

Thông báo quan trọng

Hệ thống phát hiện rằng tất cả đặc trưng âm thanh trong database đều có giá trị 0. Điều này có nghĩa là dữ liệu chưa được training để trích xuất đặc trưng thực tế từ các file audio.

Để sử dụng tính năng so sánh giọng nói, bạn cần chạy training trước.

Hướng dẫn Training
Bước 1: Cài đặt dependencies
pip install -r requirements.txt
Bước 2: Chạy training
python simple_training.py
Bước 3: Kiểm tra kết quả
python debug_voice_comparison.py
Bước 4: Khởi động lại Web App
python run_web_app.py
Giải thích

Training là gì?

Training là quá trình trích xuất các đặc trưng âm thanh thực tế từ các file audio trong folder trainset/ và lưu vào file super_metadata/trainset.csv.

Tại sao cần training?

Để so sánh giọng nói, hệ thống cần có dữ liệu đặc trưng thực tế của từng file audio. Hiện tại tất cả đặc trưng đều là 0, nên không thể thực hiện so sánh có ý nghĩa.

Training sẽ làm gì?

  • Đọc từng file audio trong trainset/
  • Trích xuất 17+ đặc trưng âm thanh (pitch, MFCC, tempo, etc.)
  • Lưu kết quả vào super_metadata/trainset.csv
  • Cho phép Web App thực hiện so sánh giọng nói
